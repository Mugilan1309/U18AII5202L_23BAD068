# Experiment 6: High-Dimensional Data Visualization

## Description
This experiment applies advanced techniques to visualize high-dimensional retail business data. It employs Parallel Coordinates, Bubble Charts, and Trellis Displays to analyze sales, profit, and discount patterns across different regions and customer segments.

## Files Included
*   `code/ex6.R`: R script implementation of the visualizations.
*   `data/6.retail_business.csv`: Retail transaction dataset.
*   `figures/`: Directory for output visualizations.

## Visualization Outputs
1.  **Parallel Coordinates Plot:**
    *   **Objective:** To compare Sales, Profit, and Discount across Customer Segments.
2.  **Bubble Chart:**
    *   **Objective:** To visualize the relationship between Sales and Profit, with Discount represented by bubble size.
3.  **Trellis Display:**
    *   **Objective:** To show performance trends (Sales vs. Profit) faceted by Region.

## Dependencies
*   `ggplot2`
*   `GGally`
*   `dplyr`
