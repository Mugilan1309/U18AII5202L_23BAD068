# Experiment 5: Data Visualization with Power BI

## Description
This experiment demonstrates the use of Microsoft Power BI for interactive data visualization. It focuses on creating detailed reports to analyze student performance data using Power BI's drag-and-drop interface.

## Files Included
*   `pbix/ex5.pbix`: The Power BI project file containing the interactive reports and dashboards.
*   `data/5.student_performance.csv`: The source dataset used for generating the reports.

## Visualization Outputs
The Power BI report includes:
*   Interactive dashboards analyzing student grades and performance metrics.
*   Visualizations such as bar charts, pie charts, and slicers for filtering data dynamically.

## Prerequisites
*   **Microsoft Power BI Desktop:** Required to open and view the `.pbix` file.
